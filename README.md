# Software Projekt - Labyrinth

## Um was geht es?

Das sog. Software Projekt ist ein Modul der ZHAW, an welcher wir studieren.
Es handelt dreht sich um die Erarbeitung von Wissen im Zusammenhang mit einem Stück software.

In diesem Software Projekt geht es um verschiedene Algorithmen, welche in der Lage sind, ein Labyrinth zu lösen. Die Algorithmen sollen miteinander verglichen und Rückschlüsse gezogen werden.
